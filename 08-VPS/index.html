DigitalOcean

DigitalOcean官网： https://www.digitalocean.com/

DigitalOcean 是一家口碑不错的 IDC 商家，总部位于美国，在世界各地有数据中心，租用服务器方便，可按小时收费。
可利用 Floating IP 无限换 IP
这家最便宜的套餐是 5 美元每月，配置如图：

1. 注册
要注意的是要使用他家的服务需要以下二选一来进行认证：

  • Paypal 充值 5 美元
  • Visa 或 Master 信用卡（无需充值）

上面的官网链接可以直接注册，或者使用我的邀请链接你可以获得 10 美元的账户余额（此时我没有收益，除非你在do花费了25美元，我才能获得25美元的收益）:邀请链接

注册流程：

  1. 输入你常用的邮箱（因为以后登录经常需要输入验证码）和密码
  2. 点击 Create Account
  3. 打开你收到的确认邮件，点击邮件中的链接
  4. 通过 Visa 信用卡（会短暂收取 1 美元的授权验证，但不是真的扣款）或 Paypal 充值认证
  5. 使用

2. 创建 VPS

DigitalOcean 将 VPS 称为 Droplet ，倒是很应景。
流程：

  1. 在 这个链接 页面上 点击 Create - Droplet
  2. 选择系统（推荐Ubuntu 16.04 x64/Debian 9.5 x64/Centos 6.x 7.x x64）
  3. 选择 $5/mo 的套餐
  4. 国内建议选择 旧金山（San Francisco 2 ） 或者 新加坡（Singapore）数据中心
  5. 勾选可选功能（ipv6 Monitoring）
  6. 把主机名改得简短些，点击 Create ，一封包含服务器用户名和密码的邮件会发送到你的邮箱
  7. 一两分钟后你的 VPS 就可以创建好，点击记录ip地址，使用 这个链接 里的 SSH 及SFTP 软件连接主机使用

使用（日后补上）：

  • 搭建 $$ $$R V2rａy Br0Ok 等 代理服务器 实现 CrossTheGreatFireWall
  • 配置 LNMP LAMP 等环境，搭建网站、个人博客等
  • 搭建个人（离线）网盘
  • 等等
