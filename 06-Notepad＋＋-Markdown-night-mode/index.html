06 打开Notepad++夜间模式并添加Markdown语法高亮和实时预览插件

  1. 2018-07-12
  2. 465字
  3. 2 min read

增强 Notepad++ 的功能和使用体验。

一、前言

Notepad++ 是一个功能强大的文本编辑器，支持多种文件格式，打开速度也很令人满意。
最近开始使用 Markdown 来写作，发现 Notepad++ 没有提供 Markdown 支持。想到 Notepad++ 支持插件扩展，我就打开了搜索引擎——果然有。
另外，长时间面对白色背景有点刺眼。我找了半天没找着换主题的地方，经过一次搜索——也找到了。

二、夜间模式

操作方法：

工具栏 -> 设置 -> 语言格式设置 -> 选择主题


下拉项中多数主题是看着眼睛比较舒服的。我个人偏爱 Zenburn 这个主题。
默认字体对中文阅读不好，偷懒的更换办法是勾选图中的 使用全局字体 。

三、添加 Markdown 语法高亮和实时预览插件

1. 添加语法高亮

Notepad++ 默认不支持 Markdown 语法高亮，所以得自己给它添加支持。
找到一个 Github 项目： markdown-plus-plus
点击下载： markdown-plus-plus-beta.zip
解压到你想要的地方， 4 个文件分别对应 4 个主题。
按照以下顺序操作：

工具栏 -> 语言 -> 自定义语言格式 -> 导入 -> 选择你喜欢的主题的对应 .xml


重启 Notepad++ ，完成。

2. 添加实时预览插件

操作方法：

工具栏 -> 插件 -> Plugin Manager -> show Plugin Manager -> 选中列表中的 MarkdownViewer++ -> 点击 Install -> 根据提示选 是 重启 Notepad++


如果你没有 Plugin Manager ，那么需要先安装 Plugin Manager 插件。
点击下载： PluginManager_v1.4.11_UNI.zip
解压覆盖到 Notepad++ 安装目录，如： C:\Program Files (x86)\Notepad++ 或 C:\Program Files\Notepad++
重启 Notepad++ ，按照上述操作方法操作，完成。
