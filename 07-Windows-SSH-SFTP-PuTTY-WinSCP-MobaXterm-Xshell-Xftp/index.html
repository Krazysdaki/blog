一些顺手的工具。

第 07 篇

~

这篇按自己接触的顺序写，主要是给自己做备忘。

一、 PuTTY

一个 SSH 软件，有汉化。
一开始接触 VPS 的教程一般都推荐这个软件，简单易用，大众工具，一搜即有。
主要优点：

  • 粘贴方便：点击右键直接粘贴；
  • 复制方便：自动复制选中内容（有时候也会造成麻烦）；

缺点：

  • 密钥为 .ppk 格式，须用 PuTTYgen 进行转换。

单文件，小巧方便，功能俱全。

二、 WinSCP

一个 SFTP 软件，有汉化。
同上。
主要优点：

  • 操作接近 Windows 资源管理器；
  • 右键菜单强大，可配置权限、执行命令、直接压缩解压等；
  • 可直接打开 PuTTY 、可导入 PuTTY 会话；
  • 等。

缺点：

  • 传输的文件会变成 Windows 格式（本应是 UNIX 格式），一些脚本会因此报错无法执行，摧毁小白；
  • 与 Notepad++ 配合不佳（有几率报 文档被另一程序编辑、文档不存在 等问题）；
  • 等。

三、 MobaXterm

一个 SSH、SFTP 结合的软件，无汉化。
功能强大，我用不完。
主要优点：

  • 整合 SSH、SFTP ，方便快捷；
  • 多终端窗口并列；
  • 可导入 PuTTY 会话；
  • 可直接用作 HTTP、FTP 等服务器；
  • 等。

缺点：

  • 右键菜单与 WinSCP 相比略弱；
  • 复制粘贴（右键选择菜单过长）略显麻烦；
  • 等。

四、 Xshell 和 Xftp

都有汉化，官网（以及国内代理商官网）下载试用版需要填写一些信息，不太方便。
可以在网上找绿色版或者直接找安装包，应该可以免费无限试用。我在这里提供两个版本：（点击下载）

[Xshell 6.0.0089.zip 绿色破解版](https://dl.knect.win/Software/Windows/Xshell 6.0.0089.zip)

xftp6-v6.0.0085.zip 安装包

前者与 PuTTY 同为大众程序，名声在外，不过有曝光过后门漏洞。

Xshell 复制方便（浮标单击），粘贴与 MobaXterm相同。密钥文件可直接用 .pem 格式。

Xftp 右键不强，有文件夹同步功能，有利于工作环境的一致性。

两者配合方便，可互相打开会话，但不兼容 PuTTY 系会话。
我目前将这两个作为主力。

这些工具将会陪着我探索 Linux 、网络服务器这些未知世界。
