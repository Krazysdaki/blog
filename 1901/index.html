第 01 篇文章

前言

今天是 2019 年的第一天，按前两年的惯例，我应该 Check 一下去年的完成情况，然后给出新一年的 Wish & To-do List。
然而去年的完成情况实在惨淡，我都不忍整理，所以就随他去吧。
今天草草在纸上写了下新一年的期望，发现问题很大：想做的事情太多，而能控制的时间太少。
其实常是好高骛远，去年列的读书观影目标那么少，都没能完成，新的一年甚至还想给自己加倍，确实是贪多了，考虑减量或放弃一些项目。

初始版本

一、学习生活

  1. 学业：课程均分达到 80
  2. 英语：努力备考四级，争取 600+
  3. 记账：把花销记清楚
  4. 健身：体重减下来，体能体型升上去
  5. 兼职：再做几个月收银员，然后该换就换

二、技能训练

  1. 编程开发：看完 C++ 教程，熟悉 C++ 开发流程，尝试学习 java 和 Android 开发
  2. 练习写作：写博客，结合输入（读书观影）进行输出，每周 1~2 篇，年度目标为 100 篇
  3. 学习乐器：坚持练习，并在年内争取开始学习双管陶笛

三、内容输入

  1. 读书：每周一本，年度目标为 50 本
  2. 观影：目标为 100 部，剧集每季折算 1 部

删减后的版本

要完成以上内容，除非我的一天有 40 个小时……
删减是势在必行的，发现少了还可以根据情况补充，所以其实问题不大。
然而发现能删减的其实不多，所以把读书观影和写作减量了，如下。

一、学习生活

  1. 学业：课程均分达到 80
  2. 英语：努力备考四级，争取 600+
  3. 记账：把花销记清楚
  4. 健身：体重减下来，体能体型升上去
  5. 兼职：再做几个月收银员，然后该换就换

二、技能训练

  1. 编程开发：看完 C++ 教程，熟悉软件开发流程，尝试学习 java 和 Android 开发
  2. 练习写作：写博客，结合输入（读书观影）进行输出，每周 1~2 篇
  3. 学习乐器：坚持练习，并在年内争取开始学习双管陶笛

三、内容输入

  1. 读书：年度目标为 35 本
  2. 观影：目标为 70 部（剧集每季折算 1 部）

后记

这波删减几乎没减多少，无所谓了，有个盼头就行。
希望新一年活的不那么行尸走肉~
其实也了解到惰性是很难战胜的，想要每一天都兼顾充实和快乐几乎不可能，但我想找到一个平衡点，保持自己的成长。
过去一年多新的成长太少了，我不希望上了四年大学之后，还和四年前一样愚蠢。
