前言

报名了九月份的全国计算机二级考试，今天准备安装考试环境—— Visual C++ 2010 Express ，即学习版。意识到这个东西好像是免费的，就不用找什么密钥了。
过程中发现有一些好玩的东西，便记录下来，以备查阅。

下载

提供两个方法，按需选择。

方法一（最便捷）

自然就是—— MSDN, I tell you 啦！

进入这个网站，左侧目录选择 开发人员工具 ，往下翻选中 ·Visual Studio 2010·
然后在右侧选择 中文-简体 ，找到 Visual Studio 2010 Express
点击文件名尾部的 详细信息 ，将一串以 ek2k:// 开头的链接复制下来
打开迅雷新建下载，就 OJBK 啦。

这个网站提供了几乎全部的微软镜像，包括 Windows 桌面系统、服务器系统和 Office 套件等，未探索过的同志可以自行探索。

方法二（走官方）

我发现在微软注册为开发者，并免费加入 Visual Studio Dev Essentials 计划，就可以下载到很多微软的开发工具。
并且这个计划还包含了一些别的好玩的东西，自行探索~

若没有微软账号，请注册一枚。

访问此处： Visual Studio Dev Essentials ，

点击 立即加入或访问
要求填写一些个人信息，随便填也行~

然后访问此处： 下载旧版 Visual Studio 软件 | Visual Studio

可以看到这个网页上有很多版本的 Visual Studio
往下找到 Visual Studio 2010 和其他产品 ，点击右侧的 下载
找到 Visual Studio 2010 Express 选中 English
选择下拉列表中的 Chinese-Simplified （一般是第一项）
点击 Download ，也 OJBK 啦。

一个门槛有点高的大文件下载提速小技巧：用 VPS （一般带宽都很大）高速下载后，再高速下到本地系统，即搭建一个个人离线网盘。
相关过程日后补上。（疯狂挖坑）

安装和注册

下载到我们手里的是一个 ISO 镜像文件，可以直接解压到合适的地方，或者挂载为虚拟光驱。
可以看到这个镜像里包含了 C++ 、 C# 、 VB 、等环境，但这次我要安装的是 Visual Studio 2010 Express
于是进入 加载的光驱或解压文件夹/VCExpress/ ，找到 setup.exe 双击打开按步骤安装即可。过程如有不懂可以全部默认。

安装完成后，从开始菜单打开软件，
依次点击 菜单栏-帮助-注册产品 弹出的窗口提示试用剩余 29 天，需要注册并且注册免费。
打开给出的网页，发现要填一大堆信息……
好吧，为了完完全全是正版，我填！
